Elemen HTML sebaris

HTML mendefinisikan daftar panjang tag inline yang tersedia, daftar lengkap yang dapat ditemukan di Mozilla Developer Network.

  • Untuk menebalkan teks, gunakan <strong>.
  • To italicize text, use <em>.

  • Kutipan, seperti — Mark Otto, harus menggunakan <Mengutip>.
  • Deleted teks harus menggunakan <del> dan Dimasukkan teks harus menggunakan <ins>.
  • Penggunaan teks superskrip <sup> dan penggunaan teks subskrip <sub>.
